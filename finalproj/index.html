Font Rendering

By Yu Heng Cheng, Kareem Koujah, Waleed Latif, Zachary Zollman

Summary

  • Provide us a descriptive title, 2-3 sentences that summarize your project, and list your three team members.

We are planning to write a Bezier Curve rasterizer to display TrueType files. First, we are going to do a CPU based text rendering system but if we have the time, then we will attempt to speed this up and create a GPU based text rendering system.

Problem Description

  • Here you should provide the context for your idea. Describe the problem that you are trying to solve, why it is important, where it is challenging. Give us a general idea on how you are going to solve it.

The task we are implementing is important because font rendering is one of the most common tasks done by our computers. The reason we are interested in this particular task is because we were all interested in the part of the class going over rasterizing, and this weaves together concepts from that segment as well as concepts like Bezier curves. We anticipate the task to be challenging because we will need to interpret and understand the TrueType files and process them into a format that we can work with. Moreover, though it seems like using bezier curves to map the outline of text to render fonts and then rasterize them seems trivial, it is actually complicated because of the unfamiliar type of file (.ttf) that we are working with. We plan to accomplish this by first ideally using a library to parse the .ttf files to create the mesh that we are familiar with, then plotting control points which are used to create bezier curves and map the fonts. We will have to make some key decisions on quadratic vs cubic bezier curves, and we will do this when we see either option’s effects on performance and accuracy.

We will measure quality by adherence to the intended appearance of the glyphs, ability to customize the output, and the time it takes to render fonts with complex glyphs. Ideally we will have implemented GPU rendering via NVIDIA’s tools and be able to render quickly.

What questions do you plan to answer with your analysis?

We wish to discover how the TrueType binary font file encodes the Bezier curves and how to extend the Bezier curves rendering into creating shapes at different sizes and variations. Furthermore, using hints to adjust the final bitmaps into a recognizable shape at low resolution is another key aspect we want to explore. Applying GPU acceleration is the final goal that we want to deliver to demonstrate its application beyond typical 3D graphics.

Baseline: is to support CPU-based rendering of text based on TrueType font to a file for any string consisting of characters supported by that font.
